Proyecto Escritorio 2

Proyecto inspirado en la empresa de galletas Crumbl Cookies, esta aplicación es de uso administrativo ya que permite realizar diferentes operaciones (CRUD), la primer función es un inicio de sesión, si es un empleado permite ingresar datos de ventas, de produtos, de proveedores, también permite descargar un reporte en excel y guardar cada registro en una base de datos relacional. Lenguajes: Java y MySQL
