Proyecto Web 1

Proyecto inspirada en la marca de ropa Zara, te permite seleccionar productos y la cantidad deseada y agregarlos a un carrito de compras. Se maneja por sesiones, por lo tanto se se inicia desde otro navegador tendrá que ingresar productos al carrito. Permite eliminar productos del carrito, registrarse e iniciar sesión y realizar la compra.Lenguajes: HTML, PHP y CSS con conexión a base de datos SQL.
